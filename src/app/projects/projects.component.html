Mars Rover
My solution to a coding challenge to create a command line interface that emulates communications between a remote user and a rover landing on a isolated plateau on Mars. The user can provide various commands to navigate the Mars Rover, and the software will provide heading and location details back to the user.
Environment
C#, Visual Studio
